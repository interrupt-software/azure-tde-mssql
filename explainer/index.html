Unauthorized User

Weak authentication and insecure storage of credentials leads to account takeover and impersonation.

If valid users hold privileges that exceed the requirements of their job function, these privileges may be abused by an attacker who compromises their account.

Unauthorized Access

With account impersonation, bad actors can install Remote Access Tools (RATs) to target server-side data stores.

After attackers target a system's potential vulnerabilities, RATs can download and execute malware of choice.

MS SQL Servers

Data stores that are unencrypted fail to protect confidentiality and integrity of sensitive data.

With access to a system, attackers can compromise an organisation by stealing raw data, deleting data, or deploying ransomware encryption on the data files.

Unencrypted Data

Past the environment boundary protection, Remote Access Tools (RATs) can syphon raw data in stealth mode while falsifying audit trails.

RATs target servers and their databases as well as user machines. Encryption keys must not fall into the wrong hands but must be available when needed to restore data.
